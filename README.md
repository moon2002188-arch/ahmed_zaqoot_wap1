# منصة غزة للمساعدات الإنسانية - Gaza Aid Platform

مشروع جامعي نهائي.

##  فكرة المشروع (Problem → Solution)
**المشكلة:** بعد الحرب، يعاني سكان مدينة غزة من تشتت جهود الإغاثة وصعوبة معرفة أماكن توزيع المساعدات (الغذاء، الدواء، المياه) ومواعيدها الدقيقة، مما يؤدي إلى الفوضى وضياع الفرص على المحتاجين.

**الحل:** إنشاء منصة ويب مركزية وبسيطة تمكن السكان من معرفة نقاط التوزيع المحدثة لحظياً، والبحث عنها حسب المنطقة أو النوع، وتقديم طلبات استغاثة واضحة للجهات المعنية.

##  الفئة المستفيدة
1. **سكان قطاع غزة المتضررين:** للبحث عن المساعدات وتقديم الطلبات.
2. **الجمعيات الخيرية وفرق الإغاثة:** لتنظيم البيانات وتوجيه الموارد للأماكن الأكثر احتياجاً (مستقبلاً).

##  أهم الميزات
- **عرض بيانات المساعدات:** جدول تفاعلي يوضح نوع المساعدة، المنطقة، والحالة (متاح/مكتمل).
- **البحث والفلترة:** إمكانية البحث الفوري باسم الحي أو نوع المساعدة دون إعادة تحميل الصفحة.
- **نموذج طلب مساعدة:** نموذج بسيط وسريع للمواطنين لطلب إغاثة عاجلة مع التحقق من صحة البيانات.
- **تفاعلية كاملة:** استخدام JavaScript & jQuery لتحديث الحالة وعرض التفاصيل في نوافذ منبثقة (Modals).
- **تصميم متجاوب:** يعمل بكفاءة على الهواتف المحمولة والأجهزة اللوحية والحواسيب.
- **الوضع الليلي (Dark Mode):** لتوفير طاقة البطارية وراحة العين، مع حفظ الإعدادات.

##  التقنيات المستخدمة
- **HTML5:** لبناء هيكل الصفحة الدلالي.
- **CSS3:** للتصميم والتنسيق المخصص (Animations, Gradients).
- **Bootstrap 5:** لسرعة التصميم وضمان التجاوب (Grid System, Components).
- **JavaScript (ES6):** للمنطق البرمجي والتعامل مع البيانات.
- **jQuery:** لتسهيل التعامل مع DOM والأحداث (Events) والتأثيرات الحركية.
- **Font Awesome:** للأيقونات التوضيحية.

##  طريقة التشغيل
1. قم بتحميل ملفات المشروع (أو استنساخ المستودع).
2. تأكد من وجود اتصال بالإنترنت لتحميل مكتبات Bootstrap و jQuery (CDN).
3. افتح ملف `index.html` في أي متصفح حديث (Chrome, Edge, Firefox).
4. استعرض الأقسام المختلفة وجرب التفاعل مع الجدول والنموذج.

##  هيكلية الملفات
- `index.html`: الصفحة الرئيسية للمشروع.
- `css/style.css`: ملف التنسيقات المخصص.
- `js/main.js`: ملف الأكواد البرمجية (JavaScript/jQuery).
- `assets/`: مجلد الصور والوسائط.
